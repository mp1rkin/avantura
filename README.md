# Jump King Clone

2D платформер в стиле Jump King, созданный на Godot 4.6.3

## Управление
- **A/D** или **Стрелки влево/вправо** - движение
- **Пробел** - зарядка и прыжок (удерживай для зарядки, отпусти для прыжка)

## Цель игры
Прыгай по зеленым платформам и доберись до красной финишной точки наверху!

## Запуск
Скачай `.exe` файл из папки `export/` и запусти его.
